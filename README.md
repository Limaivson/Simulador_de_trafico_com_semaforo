# Sinais GenIAis - Gerenciamento Inteligente de Tráfego
Este projeto, consiste de gerenciamento inteligente de tráfego utilizando algoritmo genético para 
otimização do fluxo de veículos em uma cidade. Para validação inicial de dados, foi utilizada uma simulação em Python.

Este protótipo foi desenvolvido durante a hackathon [Hacker Cidadão 10.0](http://hackercidadao.rec.br/).
Para saber mais sobre o evento, acesse o site.

## Funcionalidades
- Simulação de tráfego em uma cidade
- Algoritmo genético para otimização do fluxo de veículos
- Implementação de semáforos inteligentes para reduzir congestionamentos
- Interface gráfica para visualização dos resultados da simulação
## Tecnologias utilizadas
- Python
- Bibliotecas NumPy, Matplotlib e Pygame
